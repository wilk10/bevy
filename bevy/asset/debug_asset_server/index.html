DebugAssetServerPlugin

Facilitates the creation of a “debug asset app”, whose sole responsibility is hot reloading assets that are “internal” / compiled-in to Bevy Plugins. Pair with load_internal_asset to load “hot reloadable” assets The debug_asset_server feature flag must also be enabled for hot reloading to work. Currently only hot reloads assets stored in the crates folder.
